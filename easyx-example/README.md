# easyx-example

EasyX-RS库的示例代码集合，展示了EasyX-RS的各种功能和用法。

## 项目概述

本项目包含了多个使用EasyX-RS库的示例代码，展示了库的各种功能，包括：

- 基础图形绘制
- 鼠标事件处理
- 键盘事件处理
- 消息循环机制
- 文本渲染

## 示例内容

### main.rs - 基本事件处理示例

该示例展示了如何使用EasyX-RS库处理鼠标和键盘事件，主要功能包括：

- 创建800x600像素的图形窗口
- 显示提示信息和退出说明
- 实时显示鼠标位置
- 显示按键的虚拟键码
- 支持按ESC键退出程序

## 构建和运行

### 构建

```bash
cargo build
```

### 运行

```bash
cargo run
```

## 使用说明

1. 运行示例程序后，会显示一个图形窗口
2. 移动鼠标，窗口会实时显示鼠标的坐标位置
3. 按下任意键，窗口会显示按键的虚拟键码
4. 按ESC键可以退出程序

## 技术要点

- 使用`run_flags`函数创建带有特殊标志的窗口
- 使用`peek_message`函数非阻塞获取消息
- 使用`out_text`函数渲染文本
- 使用`text_width`和`text_height`函数获取文本尺寸
- 使用`fill_rectangle`函数清除旧的显示内容
